# easy-pdf-watcher
Custom Windows tool for PDF detection, work order extraction, file renaming, and export routing.
-----
Easy PDF Watcher

Setup
1. Open:
   .vbs\setup_pdf_easy_watcher.vbs

2. Choose:
   - the folder you want to watch for PDF files
   - the folder where renamed PDFs should be exported

3. Click Apply to save your settings.
   If you click Cancel, both folders will default to Documents.

Startup
1. Press Win + R
2. Type:
   shell:startup
3. Create a shortcut of:
   .vbs\main_pdf_easy_watcher.vbs
4. Move that shortcut into the Startup folder

Done
The watcher will now start automatically when you sign in to Windows.
